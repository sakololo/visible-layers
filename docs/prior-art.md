# Prior Art And Project Positioning

Visible Layers should be presented carefully. The goal is not to claim that automatic layer decomposition does not exist. Recent research and tools are making that possible.

## What Exists

Current work includes:

- transparent image and layer generation
- single-image anime layer decomposition
- occluded-region inpainting
- material-separation helpers for existing art workflows
- PSD or PSD-like layer exports in some tools

These are important advances and should be treated as prior art, possible integrations, and inspiration.

## What Remains Hard

Automatic decomposition does not fully replace a Live2D production workflow. A usable rigging-preparation workflow still needs:

- artistically meaningful part separation
- consistent naming and categories
- layer-order inspection
- hidden-region and overdraw checks
- boundary cleanup
- repair masks and prompts
- human quality control
- later rigging, physics, and motion work

## Visible Layers Position

Visible Layers sits between automatic decomposition and production rigging.

It focuses on:

- metadata
- previews
- validation reports
- gap detection
- repair workflows
- interoperability
- repeatable examples

The project should be described as early-stage, exploratory, and human-in-the-loop.
